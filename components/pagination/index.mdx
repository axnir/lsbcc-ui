## API

|    属性   | 说明     | 类型            | 默认值 |
| -------- | -------- | -------------- | ------ |
|  total   | 数据总数 |     number      |    0   |
| pageSize | 每页条数 |     number      |    -   |
| current  | 当前页数 |     number      |    -   |
| onChange| 页码改变的回调，参数是改变后的页码 | function (page) | - |
