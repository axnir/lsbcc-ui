## API

|    属性   | 说明     | 类型            | 默认值 |
| -------- | -------- | -------------- | ------ |
|  title   | 卡片标题 |     string      |    -   |
|  content | 卡片内容 |    ReactNode    |    -   |
| placement  | 卡片位置 |  "top"/"right/"bottom"/"left" |    'left'   |
